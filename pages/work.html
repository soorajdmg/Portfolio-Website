IRCTC eTicket Booking Portal IRCTC eTicket Booking Portal is a simple and efficient Python-based application made to
book train tickets, order food, locate the trains, cancel the already booked tickets, and
also do inquiries about the PNR number of your booked tickets of the Indian Railways.
The frontend is made using Python and the backend is made using MySQL.
