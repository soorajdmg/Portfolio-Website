Personal Projects

IRCTC eTicket Booking Portal IRCTC eTicket Booking Portal is a simple and efficient Python-based application made to book train tickets, order food, locate the trains, cancel the already booked tickets, and also do inquiries about the PNR number of your booked tickets of the Indian Railways. The frontend is made using Python and the backend is made using MySQL. Finance Tracker Finance Tracker is a personal project I developed to manage and track my own finances effectively. It uses Google Sheets for data visualization, making it easier to track income, expenses, and budgets. This project underscores my skills in data analysis, application development, and personal finance management. Password Manager Password Manager is my inaugural Windows application, designed with a strong focus on security and user-friendliness. It empowers users to efficiently organize and protect their passwords and sensitive data. With strong encryption, seamless password generation, and an intuitive interface, it simplifies password management, making it an ideal choice for my first Windows app.
